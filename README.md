# Mini E-Commerce Project

## Description
Ce projet est une application web de mini e-commerce qui permet aux utilisateurs de consulter des produits et de les ajouter au panier.

## Fonctionnalités
- Voir la liste des produits
- Voir les détails d’un produit
- Ajouter un produit au panier
- Supprimer un produit du panier

## Technologies utilisées
- React.js
- Node.js
- Express.js
- MongoDB
